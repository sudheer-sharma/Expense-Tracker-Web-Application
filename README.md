# 📊 Expense Tracker Web Application

A responsive Expense Tracker Web Application built using HTML, CSS, and JavaScript. This project helps users manage their monthly income and expenses with real-time calculations, data visualization, and Excel export functionality.

---

## 🚀 Features

- Add and delete expense records dynamically  
- Real-time calculation of total expenses and remaining salary  
- Interactive pie chart visualization using Chart.js  
- Export expense data to Excel using XLSX library  
- Responsive design for mobile and desktop devices  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript (ES6)  
- Chart.js  
- XLSX Library

## 💡 How It Works

1. Enter your name and monthly salary  
2. Add expense items with amount  
3. View all expenses in a dynamic table  
4. Track total spending and remaining balance  
5. Visualize data using pie chart  
6. Export data to Excel file

## 📊 Key Functionalities

- CRUD Operations (Add & Delete Expenses)  
- Real-time UI updates  
- Data visualization using charts
